# `dedup.ui.components`

- **`toast_manager.py`** — used by **`CerebroCTKApp`** (toasts).
- **Other modules** — classic **ttk** widgets and review helpers retained for **tests** and **`ReviewVM`** / projection glue. They are **not** required for the CTK shell at runtime.

Prefer importing the specific module you need rather than this package’s namespace.
